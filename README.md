# Aether - Next.js E-commerce Template

Thank you for purchasing Aether.

For complete instructions on how to install, customize, and deploy this template, please refer to the **Documentation** folder included in your download package.

## Quick Start

1.  Navigate to the `Template` directory.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.

We hope you love building with Aether!
